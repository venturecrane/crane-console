# Reviews

Code reviews, platform audits, and retrospectives. These capture findings, recommendations, and lessons learned from structured evaluations of the codebase and infrastructure.

## Who This Is For

**Running a platform audit?** Check recent reviews for prior findings and patterns. Run `/platform-audit` for a fresh assessment.

**Investigating an incident?** Retrospectives document what happened, root causes, and corrective actions.

**Reviewing code quality?** Code reviews capture cross-cutting concerns and recommendations across ventures.
