# Documentation

| Directory    | Purpose                                             |
| ------------ | --------------------------------------------------- |
| `adr/`       | Architecture Decision Records (numbered, immutable) |
| `design/`    | Design briefs, specs, governance, strategy          |
| `handoffs/`  | Historical session handoff records                  |
| `infra/`     | Fleet inventory, secrets, networking                |
| `pm/`        | Product requirements                                |
| `process/`   | Team workflows, procedures, agent briefs            |
| `research/`  | Technology evaluations and research spikes          |
| `memory/`    | Enterprise memory governance (`governance.md`)      |
| `runbooks/`  | Operational how-to guides                           |
| `standards/` | Engineering standards and templates                 |

## Blog content

Articles and build logs live in [vc-web](https://github.com/venturecrane/vc-web)
(`src/content/articles/` and `src/content/logs/`). Blog concept issues
are tracked here with the `content:blog` label.
