# Kid Expenses

**Tagline:** Expense tracking for co-parents

## What It Is

Shared custody means shared costs and shared disagreements about money. Kid Expenses gives divorced and separated parents a structured way to log, split, and settle child-related expenses without the conflict.

## Tech Stack

Next.js, Cloudflare Workers, D1

## Operator Model

### Captain's Domain

- Product direction: which features get built, what gets cut
- Launch timing and pricing decisions
- User-facing policy decisions (what data is stored, how disputes are handled)

### Agent Execution

- Feature development and PR execution
- Testing and QA
- Customer support triage
- Deployment and infrastructure maintenance

### The System

The franchise prototype is the expense-sharing workflow: log, split, settle. Once live, agents handle the full operational loop - support queues, billing, deployment - while the Captain owns the product roadmap and launch calls.
