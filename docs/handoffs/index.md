# Handoffs

Session handoff records - point-in-time artifacts documenting what was accomplished, what's in progress, and what needs attention next. These provide continuity between agent sessions.

## Who This Is For

**Picking up work from a previous session?** Check the most recent handoff for context on where things left off.

**Investigating a past decision?** Handoffs capture the reasoning and state at the time of each session transition.

Note: Active session handoffs are managed by the `crane_handoff` MCP tool and stored in D1. The files here are historical records from early sessions before the automated handoff system was established.
